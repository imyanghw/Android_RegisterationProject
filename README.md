# Android_RegisterationProject
> 안드로이드 수강신청 앱 프로젝트

## 개발환경 및 사용버전
- Android Studio(Java)
- SdkVersion 29(Android 10.0)
- 테스팅 환경(Galaxy Note9)
- cafe24 웹호스팅 

## 구현내용
- 로그인/회원가입(아이디중복체크)
- 어플리케이션 정보(팝업창)   
- 공지사항, 강의목록, 시간표, 강의분석
- 학부, 년도, 학기, 전공에 따른 강의목록 조회
- 추가한 강의에 대하여 시간표를 나타냄
- 추가한 강의에 대하여 삭제기능      
- 강의 신청인원, 강의 경쟁률을 나타냄
